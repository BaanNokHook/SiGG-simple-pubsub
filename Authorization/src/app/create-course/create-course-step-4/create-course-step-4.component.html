Issuance Initiation-Same-device

This step is used to pass available information relevant for the credential issuance to ensure a convenient and secure process.

Trigger the credential issuance initiation:

The wallet must be able to open the link starting with https://qr-open.it/g/4ceBkGgbQ6l

The wallet is not supposed to create a response.

Save the URL of credential_type for later: you will need it to request the VC.
